# Architecture Decision Records (ADR)

This directory stores architecture decisions for the Memories repository.

## Conventions

- Decision IDs: `ADR-YYYYMMDD-<slug>`
- Status values: `Proposed`, `Accepted`, `Superseded`, `Deprecated`
- One decision per file
- Cross-link related PRD/TDD sections and implementation log entries

Use `docs/templates/decision-log-template.md` for new ADRs.
