# Architecture Decision Records (ADR)

This directory stores architecture decisions for template-based projects.

## Conventions

- Decision IDs: `ADR-YYYYMMDD-<slug>`
- Status values: `Proposed`, `Accepted`, `Superseded`, `Deprecated`
- One decision per file
- Link related PRD/TDD sections and implementation log entries when available

Use `docs/templates/decision-log-template.md` for new ADRs.
